# どうぶつゆうびん

どうぶつがものを送り届けるゲーム

# ゲームシステム

- 郵便を食べるとHP回復
- 4人プレイ
  * Coop? 可能ならオンライン

# インストール

## 必要環境

  * Unity 2017.3.0f3
* Git
* Githubアカウント

## プロジェクトを落としてくる

```bash
git clone git@github.com:HibikineKage/ggj18.git
```

あとはUnityでggj18のフォルダを開けばOKです

## 開発時の留意点
基本的に自分の名前を付けたシーンで作業する。
複数シーンのマージは非常に面倒なので、誰か1人が作業するようにする。
できるだけプレハブ化し、他のシーンに持っていけるようにする。
